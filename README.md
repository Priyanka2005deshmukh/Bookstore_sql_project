# Bookstore_sql_project
# A simple and user-friendly Book Store Project designed to manage books, customers, orders, and billing. This project helps automate basic bookstore operations and keeps all records organized in one place.

# # Features

Add New Books – Enter book details like title, author, price, and stock.

View & Search Books – Easily find books by name, category, or author.

Update & Delete Books – Edit book information or remove books from the list.

Customer Management – Add customer details for purchase records.

Order / Billing System – Select books, calculate total price, and generate bills.

Clean – Simple and beginner-friendly.

## Project Goals

To build a complete and functional store management system

To understand CRUD operations

To improve logical thinking through real-time project developmen

## Applications

1-Small book shops

2-College projects

3-Beginner learning project
